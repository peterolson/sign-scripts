# Sign Language Script Conversion

This project parses and converts between the following writing systems for sign languages:

-   [SignWriting](https://en.wikipedia.org/wiki/SignWriting)
-   [HamNoSys](https://en.wikipedia.org/wiki/Hamburg_Notation_System)
-   [Stokoe Notation](https://en.wikipedia.org/wiki/Stokoe_notation)
-   [ASLwrite](http://www.aslwrite.com/)
-   [SLIPA](https://dedalvs.com/slipa.html)
-   [ASL Font](https://github.com/aslfont/aslfont.github.com)
